✅ Features:
   ✓ Lokale ADC Messung (GPIO 3)
   ✓ CAN-Bus Lenkwinkel vom Lenkung-ESP
   ✓ Gewichteter Abgleich (70% lokal, 30% CAN)
   ✓ Anomalie-Erkennung (Warnung wenn > ±20%)
   ✓ Fallback auf lokal wenn kein CAN Signal

🔄 Abgleich Algorithmus:
   if (CAN verfügbar) {
     diff = |lokal - CAN|
     if (diff > 20%) → Log Warning
     result = 0.7×lokal + 0.3×CAN
   } else {
     result = lokal  // Fallback
   }

📌 Hardware:
   GPIO 3 (Analog)    → Lokale Messung (12-Bit ADC)
   CAN-Bus (GPIO 10/11) → Lenkung-ESP Signal

Problem: ADS1115 wird nicht erkannt
─────────────────────────────────────
✗ Serial Output: "ADS1115 not found!"
✓ Überprüfen:
  1. I2C Verdrahtung SDA (GPIO 8) / SCL (GPIO 9)
  2. Pull-Ups vorhanden? (Typisch 4.7k OHM)
  3. A0/A1 Addresss-Pins auf GND gesetzt?
  4. Power 3.3V/GND korrekt?
  5. Try adresse mit I2C Scanner

Problem: Gaspedal springt wild
──────────────────────────────
✗ Gas Prozent schwankt um ±30%
✓ Überprüfen:
  1. Poti Verdrahtung (Connector 2 Pin 3)
  2. Kalibrierung: GasPedal vollständig fahren
  3. ADC Rausch: Sample Count erhöhen (default 3)
  4. Servo Anschluss überprüfen

Problem: Stromsensor zeigt Strom obwohl Motor aus
──────────────────────────────────────────────────
✗ curr_mA > 0 wenn kein Verbraucher
✓ Überprüfen:
  1. ACS770 Offset: sollte 2.5V sein (bei 0A)
  2. GND Verbindung Check
  3. Stromsensor Verdrahtung korrekt?
  4. ADC Kalibrierungswert anpassen

Problem: CAN Lenkwinkel wird nicht abgeglichen
───────────────────────────────────────────────
✗ steering.getAbglichSteerPercent() == getSteerPercent()
✓ Überprüfen:
  1. Lenkung-ESP sendet CAN Data?
  2. CAN-Bus Verbindung OK?
  3. can.dataValid == true?
  4. can.steerAngleCAN != 0?
